## NecroGest VCL - Landing Page Responsiva (Laboratorio #1)

Este proyecto corresponde al Laboratorio #1 del curso **ISW-521 Programación en Ambiente Web I** de la Universidad Técnica Nacional (UTN).

### Contexto del Proyecto
Consiste en la conceptualización de una Landing Page institucional para **NecroGest VCL**, un software de escritorio para la gestión automatizada de camposantos municipales. El sitio web promociona módulos basados en flujos de datos lineales reales:
* **Lista Dinámica Enlazada** para el registro general de difuntos.
* **Pila LIFO** para el control vertical de ataúdes en nichos.
* **Cola FIFO** para el despacho cronológico de solicitudes de exhumación.

### Tecnologías Utilizadas
* **HTML5 Semántico:** Estructura limpia evaluada bajo estándares de la W3C.
* **CSS3 Nativo:** Maquetado responsivo utilizando Flexbox para componentes lineales y CSS Grid para la distribución de tarjetas.
* **JavaScript Nativo (Web Storage):** Implementación de persistencia para mantener la preferencia del Modo Oscuro/Claro elegida por el usuario.
